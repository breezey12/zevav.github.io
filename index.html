Arithmetic Operators Continued

Operator Description Usage
** Exponentiation 
3 ** 3
% Mod 
4 % 3
// Floor Division* 
4 // 3
Exponentiation- straightforward. Mod is "what's the remainder when we divide 4 by 3?". Floor division is "How many times does 3 go into 4, excluding a remainder?"

String/Object Operators

Operator Description Example Usage
+ Concatenation 
'C' + 'U'
[] Index 
a_string[n]
[:] Slice 
a_string[3:7]
. Attribute 
a_list.append(5)
Concatenation is like addition, but for text(strings)- it combines the two pieces into one, in an intuitive way. Index operator returns the nth element of a string or list. Slice returns a slice of the string or list. Slices include the first index given and up to but excluding the second index given. The attribute operator gives us a way to access functions and variables that are specific to certain types of variables.
