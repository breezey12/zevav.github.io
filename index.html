Functions

Programming-speak for "verbs"

Some verbs we've already met: 

>>> int() # converts non-integers to integers
>>> str() # converts non-strings to strings
>>> len() # measures the length of a string, list, or dictionary

These are "built-in" functions, as they come with Python.

Functions do things;

Variables (integers, strings, lists) are things.

When you "run" a function,

you call it.

"Hey, len(), could you please measure this string for me?"

>>> len(this_string)
10

"Thanks, len(), you're the best."

The bit in the parentheses is the "parameter".

>> len(this_string)

The fun part:

You can make (define) your own functions!

Block of organized code.


# gives the length of a word and adds it to a list
word_list = []
word_from_user = raw_input("Enter a word for me to measure")
word_length = len(word_from_user)
word_list.append(word_from_user)
print word_from_user + " is " + str(word_length) + " letters long."
