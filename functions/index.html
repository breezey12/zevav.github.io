The fun part:

You can make (define) your own functions!

Why would you want to define your own function?

"Function: A block of organized, reusable code that is used to perform a single, related action."

A function is a verb, so

"Computer, when I say, 'Transform and measure a word,' what I mean is

  1. replace every 'a' or 'A' with an 'x',
  2. use it in a sentence,
  3. assign the length of the word to another variable,
  4. and tell me the word's length in plain language."

# word_stuff.py
a_word = "Australia" # create the word 
# replace every 'a' with an 'x'
print a_word.lower().replace('a','x')
# use it in a sentence
print "The only thing we have to fear is " + a_word + " itself."
# assign the length of the word to another variable
word_length = len(a_word)
# and tell me the word's length in plain language
print "'" + a_word + "' is " + str(word_length) + " letters long."
 

$ ipython word_stuff.py
xustrxlix
The only thing we have to fear is Australia itself.
'Australia' is nine letters long.

Anatomy of a Function

def

def transform_and_measure_a_word(a_word):
    print a_word.lower().replace('a','x')
    print "The only thing we have to fear is " + a_word + " itself."
    word_length = len(a_word)
    print a_word + " is " + str(word_length) + " letters long."
